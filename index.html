What is ASTC?

Adaptive Scalable Texture Compression (ASTC) is an advanced block-based lossy texture compression format developed by ARM. It supports variable block sizes from 4x4 to 12x12, allowing developers to fine-tune the trade-off between quality and memory size.

Void-Extent Blocks

Void-Extent blocks are a special feature of ASTC used to encode constant-color regions efficiently. By representing an entire block with a single RGBA color, it avoids compression artifacts and saves processing overhead. Our encoder utilizes this standard-compliant feature to generate valid ASTC files.

Hardware Decoding

Modern mobile GPUs (Mali, Adreno, Apple Silicon) support ASTC natively. This studio checks for the WebGL ASTC extension to decode complex textures directly on your GPU. If unsupported, our custom CPU decoder handles Void-Extent blocks seamlessly.
